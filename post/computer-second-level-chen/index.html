做题技巧：

一、选择题

选择题

  1. 尽可能多做题，因为二级考试的选择是来自题库的，你做的越多，肯定分数越高。
  2. 合理选择，毕竟时间不多了，没必要把每个选择都弄明白了，自己难以理解的就背过它。
  3. 绝对要相信自己的第一感觉，过多的考虑只会使自己对这道题越来越陌生。
  4. 注意选择题一旦退出，就不能再次进入，要注意不要漏答。

二、操作题

操作题

  1. 准确把握时间，恰当分配。较好的答题顺序是：先做ppt，然后做word，最后做excel。
  2. 养成良好的习惯：每做完一道题，就保存一次，以免发生意外情况。
  3. 在平时练习中尽量用Office2010，不要用WPS代替。
  4. 考试时遵循先易后难的原则进行答题。

一、基础选择题

  1. 计算机的发展、类型及其应用领域。

  2. 计算机软硬件系统的组成及主要技术指标

  3. 计算机中数据的表示与存储

  4. 多媒体技术的概念与应用。

  5. 计算机病毒的特征、分类与防治。

  6. 计算机网络的概念、组成和分类；计算机与网络信息安全的概念和防控。

  7. 因特网网络服务的概念、原理和应用。
